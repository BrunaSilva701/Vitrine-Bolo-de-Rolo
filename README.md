# Casa do Bolo de Rolo 🥣👩🏽‍🍳

## Tipo de negócio escolhido
Produção artesanal de bolos de rolo

## Problema de négocio
Atualmente, os pedidos são feitos exclusivamente pelo celular. O que gera:
- Desorganização das demandas
- Demora na realização dos pedidos e na entrega
- Dificuldade em visualizar os produtos dispiniveis e seus preços
- Não permite que o cliente visualize quanto vai pagar ao final

## Solução proposta
Site que permite aos clientes visualizar uma vitrine de produtos, assim como o valor que será cobrado
- Visalização clara dos produtos e preços
- Simulação do valor que será pago
- Menor demora
- Maior organização
